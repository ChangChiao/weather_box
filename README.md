# template-vite-vue-tailwind-v3

#### 串接中央氣象局開放資料 + 用 pinia 管理資料

### demo 網址

👉 [天氣盒](https://changchiao.github.io/weather_box/)

- **template of:** vite + vue
- **tailwindcss:** v3.x

## Started

```bash
yarn
# or `npm install`
```

---

## Develop

```bash
yarn dev
# or `npm run dev`
```

---

## Build

```bash
yarn build
# or `npm run build`
```
